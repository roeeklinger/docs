This project is a Mintlify documentation respository for our platform, Bytestack. It is a standard Mintlify project, cloned from Mintlify directly, and using the standard @docs.json file, etc. You can find the documentation for Mintlify at https://docs.mintlify.com/.

The Mintlify documentation CLI is available locally using `mint` commands such as `mint dev` to run the development server.

Bytestack brand has a very distinctive style, and we always want and aim to make sure that the documentation reflects that. We will be using the Mintlify theming capabilities to customize the look and feel of the documentation to match our brand. For this purpose, we have set a @custom.css file in the project, which you can use to add custom styles to the documentation. You can reference the Bytestack design system and colors to ensure that the documentation is consistent with our brand identity.

The Bytestack design system, colors, style and reference pages can all be found in Bytestack's website respository, which is located at @/Users/roee/Github/bytestack for you to reference and use as needed.

When working, assume that the mint dev server is already running and accessible at http://localhost:3000/, if you fail to reach it, ask me to start it for you. When working, load and use the /agent-browser skill to inspect the documentation and iterate or answer queastions as needed. This will allow you to see the changes you make in real-time and ensure that the documentation is consistent with our brand and design system.
