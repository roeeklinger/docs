## Key capabilities

### Natural language queries

You describe what you want in plain English — "How many times was our brand mentioned on Reddit this week?" — and BitKit translates that into a targeted data fetch across the sources you specify. No query syntax, no API stitching, no scraping code.

### Sentiment analysis

Every query can include automatic sentiment classification. BitKit runs an AI model over the results and labels each mention as positive, negative, or neutral, giving you a breakdown alongside the raw counts.

### Data stitching

BitKit's entity resolution engine identifies the same person, organization, or location across multiple sources and merges them into a unified profile. You get a single view of an entity even when the underlying data comes from LinkedIn, Google, Facebook, and GitHub.

### Scheduled jobs

You can schedule queries to run on a recurring cadence — hourly, daily, or weekly. Results are stored in your project workspace and can be exported to S3-compatible storage or pushed to your stack via webhooks.

### Integrations

BitKit supports a REST API, official SDKs for Python, TypeScript, and Go, S3-compatible storage for bulk exports, and webhooks for real-time event delivery. You can connect BitKit to the tools and workflows your team already uses.
